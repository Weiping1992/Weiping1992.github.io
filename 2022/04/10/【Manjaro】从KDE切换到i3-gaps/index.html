【Manjaro】从KDE切换到i3-gaps

2022-04-10 Linux Linux, i3-gaps, 折腾 0 评论 字数统计: 2.1k(字) 阅读时长: 8(分)

i3 is Window Manager

也许你会经常看到DE和WM这两个词。DE是桌面环境，在Linux系统中主流的有KDE、GNOME和XFCE。而WM指的是窗口管理器，常见的有i3wm、bspwm、dwm、awesome等等。相比于WM，DE更加系统，不仅拥有窗口管理能力，还自带一整套软件，例如终端软件、界面配置软件，让桌面环境更加易用。但是与之而来就是拥有更大的性能开销。

i3wm是一个平铺式窗口管理器。相比于层叠式窗口，在i3中所有窗口都是平铺在显示器中的，可以使用快捷键实现窗口间的来回跳跃，避免鼠标的介入，因此带来了更高的效率。但是相比于DE，i3wm的配置相对而言会更加复杂一些。

之前我的Manjaro安装的是KDE版本，为了折腾将KDE切换到i3-gaps。i3-gaps是i3wm的一个分支，可以实现窗口间的间距。

使用说明

第一次进入i3桌面，会自动生成i3的配置文件，这里选择Yes。Mod键使用Alt键（你也可以使用Win键）。生成的配置文件在~/.config/i3/config。

黑黢黢的桌面是因为还没有配置桌面壁纸，后面会介绍。先简单介绍一下i3的使用方法。

1.打开终端： $Mod + Enter键

这里的$Mod键就是之前步骤设置的Alt键。

如果你按下$Mod + Enter键之后没有弹出任何终端软件，有可能是还未安装i3支持的终端软件。支持的终端软件列表：i3-sensible-terminal。

由于我是从KDE切换过来的，已经安装了konsole，所以不存在这个问题。

2. 使用dmenu打开软件：$Mod + D

按下之后会在顶部出现一个可输入的显示条，输入firefox后按下Enter就打开了firefox软件。

3.切换窗口： $Mod + 方向键

如果界面上已经打开了多个窗口，要在窗口键来回切换就使用$Mod + 方向键。

4.切换工作区： $Mod + 数字键

类似KDE中一样，i3也支持多个Workspace。在workspace中切换使用$Mod + 数字键（例如1,2...）

以上只是最基本的用法，详细的使用方法参考官方手册。
